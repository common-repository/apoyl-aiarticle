== Description ==

基于百度大模型ERNIE-4.0-、GPT-3.5、GPT-4，通过prompt一句话标题描述智能创造一篇高质量的文章，可智能生成文章、AI一键改写内容、AI一键润色内容，为管理者提供大量参考内容。

## 插件功能

* 支持编辑器下面输入一句话生成文章，AI智能创作文章；简单来讲通过大模型一键自动生成文章内容
* 支持最新百度或OpenAI大模型
* 支持新文章内容改写，旧文章内容重写，一键AI内容改写+
* 支持新文章内容润色，旧文章内容润色，一键AI内容润色，让文章更加丰富
* 新增支持自定义大模型：如百度大模型ERNIE-4.0-Turbo-8K、gpt-3.5-turbo、gpt-4-turbo、gpt-4o
* 必须自行注册[百度大模型](https://console.bce.baidu.com/qianfan/ais/console/applicationConsole/application) 或者OpenAI，能正常访问百度或OpenAI接口
